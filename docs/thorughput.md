# Throughput

Ammount of successful operations per time unit (often seconds).

Measured in:
- Requests/sec
- MB/sec
- Transactions/sec

In the server side, we can calculate 2 types of Thorughputs
- `Write Throughput`
- `Read Throughput`

<br />

### Write Throughput
```js
Write Throughput = Number of users among DAU * Single user operation count
```

<br />

### Read Throughput
```js
Read Throughput = Number of users among DAU * Single user operation count
```
